# reprication_BPB2021
## TMF deforestation data
TMF deforestation data is processed in GEE, using commune shape file (gadm41_KHM_3).
The link to GEE code is available in "GEE_deforestation_data_extraction" text file.
The extracted data is cleaned and saved as dta format (01_deforestation_data.do).

## Reproduction
All robustness checks are conducted in 02_reproduction.do.
The reproduced results are summarized and the reproducibility dashboard is created in 03_summarize_results.do.
